# ADR 0002: Phase 4 聚焦 V1 产品化收口

日期：2026-06-18

状态：Accepted

背景：

核心招聘闭环和规则版 AI 已可用，项目进入 Phase 4 后，需要决定继续扩展 AI 能力还是优先做交付收口。

决策：

Phase 4 不再继续新增 AI 功能，优先完成 V1 产品化收口，包括管理仪表盘、演示数据、部署文档、验收测试和 README 产品化。

原因：

当前最需要的是稳定性、可演示性和交付文档，而不是继续扩展 AI 能力。

取舍：

真实 LLM Provider 接入继续后置，换取更高的发布确定性、更完整的演示链路和更清晰的交付边界。

影响：

后续开发优先级转向产品化收口；除非另行确认，不把真实 LLM Provider 接入纳入当前版本。
