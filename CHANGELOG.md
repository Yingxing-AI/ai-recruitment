# Changelog

## v1.0.0 - 2026-06-23

### Added

- 会话归档补充 2026-06-23 发布前收尾记录，固化今天的里程碑和环境风险。

### Changed

- 本地前端开发端口调整为 `3001`，避免与 Docker 正式入口 `3000` 冲突。
- 后端默认 CORS、`.env.example`、README、部署文档和验收文档同步对齐 `3000/3001` 双入口约定。
- Docker Compose 正式入口已恢复并验证可用，作为首个稳定发布基线。

### Known Issues

- `worker` 容器仍可能因宿主机 Docker 权限限制出现异常命名漂移；当前不影响 `3000` 前端和 `8000` 后端主访问链路。
